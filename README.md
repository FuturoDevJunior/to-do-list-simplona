# English

## 📋 Project Description

A modern and intuitive To-Do List application built with pure JavaScript, focusing on clean code principles and user experience. This project demonstrates proficiency in front-end development fundamentals and DOM manipulation.

### 🎯 Key Features

- ✨ Create, read, update, and delete tasks (CRUD operations)
- 📱 Responsive design for all devices
- 💾 Local storage integration for data persistence
- 🎨 Clean and intuitive user interface
- ⌨️ Keyboard accessibility support
- 🔄 Real-time updates without page refresh

## 🛠️ Technologies Used

- HTML5 - Semantic markup
- CSS3 - Modern styling with flexbox/grid
- JavaScript - ES6+ features
- LocalStorage API - Data persistence
- Git - Version control
- GitHub Pages - Deployment

```bash
# No special requirements needed! Just a modern web browser:

- Edge
```

## 🚀 Getting Started

```bash
# Clone this repository
$ git clone https://github.com/futurodevjunior/to-do-list-simplona

# Navigate to the project directory
$ cd to-do-list-simplona

# Open index.html in your browser
```
